# 🎯 Project Notes - 專案原始說明

> 這裡保留了所有專案最原汁原味的說明，沒有包裝，就是我真實的想法和開發動機。

## 🛠️ 開發者工具與實用小品

### workr
BFF (Backend for Backend) 自己用的 API 平台。

### ZeroSetup
一個 bat 讓人家裝好你的專案設置。

### whereismywindow
多螢幕開發者必備，現在焦點在哪個視窗，方便使用快捷鍵操作確認。

### Rawser
一個新的瀏覽器，開發專用不拿來逛拿來開發，但非常實驗還在開發，超不穩。

### LocalTab
開發者專用，一次看多個 port，方便 check 專案進度。

### cloudpipe
自己用的部署平台，取代 vercel 或 railway（超輕量，沒有要取代啦）。

### SameScreen
把東西快速傳給另一台電腦，以及用通訊軟體介面紀錄事情，同時開放滑鼠控制，方便多台電腦的人。

### PyClick
拿來點讓 AI 同意的，就不需要開危險模式，可擴充成按鍵精靈但並不會這樣做。

### SeeKeyboard
看到你打什麼鍵，主要是要拿來其他專案內部使用（也許吧）。

### recovermyweb
無痕視窗拯救者，讓你可以記憶 TAB 復原回來。

### isnowfriend
小七跟全家的友善時光檢查，也可以看附近店家庫存。

### MySpeedTest
測速工具不用等，沒耐心的人專用，使用 fast.com 的節點（這能說嗎）。

## 🎨 圖片與錄影工具

### ShotBoard
錄影工具，拿來上課或是錄影產品去宣傳的，可以有講稿畫布。

### Screenshot-OCR
截圖然後辨識，有串 AI 可以提高辨識率，拿來看 IG 妹子帳號在圖片中很方便。

### duckshot
超創新的截圖工具，可以直接截圖完上傳，裡面還帶有切版工具，前端開發者超實用。

### repic
最屌的看圖軟體，創新的 repic 檔案，方便你抓圖拖曳就抓，速度超快，也可以上傳檔案，絕對是最猛的。

## 📺 YouTube 相關工具

### DubTitle
雙螢幕使用者適合的 YT 字幕工具，有串 AI，可生成生字教材。

### meetube
沒有廣告且客製化功能的 YT，並且放了等化器與雙語字幕的擴充工具，手機就可以用，會繼續做更多擴充功能進去。

## 📝 部落格與筆記整理

### Evernote
通過 md 與 AI 生成技術筆記，就是一個筆記部落格模板。

### IdealFlow
以社交媒體發文為 UI 的 Kanban 軟體，適合紀錄開發點子。

### bookiee
通過 AI 生成書本的內容與雜誌內容，以及翻譯 RSS 文章，還待開發。

## 💻 CLI 工具

### winbat
整理所有好用的自定義 bat。

### cmdx
讓你使用 linux 的指令在 windows 中。

### wcmd
使用 linux 的時候也可以用 windows 指令，不需要學，懶人專用。

### typee
一個 bat 檔案讓你用不同編輯器打開。

### brozz
類似 antu7 的 broz 但是就是一個 bat 檔案而已，效果是超乾淨的瀏覽器視窗，適合截圖或是看影片。

## 🔮 Claude 設定與工具

### saveclaudenode
不要讓 claudecode 每次殺 node 殺到自己，裡面有附上 hook，也有保護的程式。

### vscode-claude-electron-fix
這就是在 vscode 裡面用 claude extension 開 electron 會遇到的問題，遇到就讓 AI 讀這個就解決了。

## 🤖 AI 相關

### meei
我自己寫程式要串 AI 就用這個，把 env 放這專案的本地後，叫 AI 讀 meei 就超好用了，不用找接口了。

### gitgpt
實驗性想寫一個 gpt 的 cli 程式，玩玩而已。

### monus
實驗性想要試試看做一個 manus 的工具看效果，也是玩玩，效果很普通。

## 🎤 聲音相關工具

### AprilVoice
一個語音轉文字的網頁工具。

### biaodian
不使用模型使用規則去幫語音轉文字加上標點，可以省時間省效能。

### Hanzi_ASR
我把我調整 funASR 弄到跟 Sherpa-ONNX 差不多快的一個成果。

## 🎨 內容創作者工具

### UPPER.COM
內容磁鐵網站，提供檔案給人下載，獲取目標客群 email。

### Qter
更好看的表單，創作者或開發者需要。

### PasteV
創作者專用的圖卡製造機，快速整理 OCR 與輸出。

### Text2Img
丟文字變成圖片可複製，社群用，很多開源字型，也提供簡單圖片編輯。

### personal-social-manager
經營 IG 或 threads 之類的帳號一覽網頁。

## 🧵 Threads 主題

### ThreadGram
打造類似 IG 介面的 threads，還沒弄爬蟲所以未完成。

### threads-capture
這個沒用了因為是 threads 之前 OG 有 bug 幫忙修正就是了，那時候 OG 圖會是發文者大頭貼傳訊息會很尷尬，我做短網址去 proxy 以及抓文章的圖。

## 🎯 作品展示

### atomicX
一個為了切版而生的框架，讓你用 css 當元件寫網頁，配合 tailwind，很有創意很大膽的專案。

### Py-FTPEZ
用 py 去實現 FTP，因為 FTP 介面很醜太陽春了，裡面有放 VScode 線上編輯器，當初工作要用 FTP 開發的。

### Nextjs-ptt-beauty
爬蟲爬 PTT 網站的美女圖，就是個練習網站。

### static-leaf-builder
這個是一個可以當 chrome 新分頁的網頁，就是自定義一些網頁 icon 按鈕的搜尋頁。
